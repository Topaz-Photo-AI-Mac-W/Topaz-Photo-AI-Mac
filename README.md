## Overview

Topaz Photo AI Professional Suite is a complete photo enhancement platform with batch processing, Lightroom integration, and autopilot for professional photography workflows.

Batch processing enhances entire shoots with consistent settings, handling the volume that professional photographers produce from events and sessions.

Lightroom and Photoshop integration operates as a plugin within existing editing workflows, applying AI enhancement without disrupting established processes.

Autopilot analyzes each image individually and selects appropriate enhancement settings automatically, handling the per-image variation that batch processing with fixed settings cannot address.

Topaz Photo AI Professional Suite is the enhancement platform for professional photographers who need AI photo processing with the batch capability, integration, and automation that professional volume demands.

## Performance & Integration

Topaz Photo AI is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Topaz Photo AI processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
